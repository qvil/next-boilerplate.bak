# Next Boilerplate

Super power Next.js boilerplate with Typescript, Styled-components, Redux, etc.

## Prerequisites

- Node.js >= 10.x (Recommended the latest version)

## Install

```
npm install
```

## Development

```
npm run dev
```

http://localhost:3000/

Dive into the project! Enough to fix the typo, translate also welcome!
